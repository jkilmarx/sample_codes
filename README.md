# Code samples from past projects

## fMRI Processing Codes
**hyperalignment_codes.ipynb** contains a jupyter notebook tutorial highlighting how to perform a hyperalignment analysis as in the following paper: 
- Kilmarx, J., Oblak, E., Sulzer, J., & Lewis-Peacock, J. (2021). Towards a common template for neural reinforcement of finger individuation. Scientific Reports, 11(1), 1065.

**hyperalignment_functions.py** contains helper functions used in the tutorial script

## EEG Analysis Codes
**analysis_tutorial.ipynb** contains a jupyter notebook tutorial highlighting the analysis of visual imagery EEG data as in the following paper:
- Kilmarx, J., Gamper, H., Emmanouilidou, D., Johnston, D., Cutrell, E., Wilson, A., & Tashev, I. (2022, February). Investigating visual imagery as a BCI control strategy: A pilot study. In 2022 10th International Winter Conference on Brain-Computer Interface (BCI) (pp. 1-6). IEEE.

**analysis_functions.py** contains helper functions used in the tutorial script

**main_realtime_nk.py** and **Processing_nk.py** show some samples of the realtime processing and classification of EEG data from the above paper.
